Modern JavaScript Syntax

ES6 introduced revolutionary features that make JavaScript more powerful and expressive. Arrow functions, destructuring, and template literals are now standard.

  • Arrow functions with implicit returns
  • Destructuring for cleaner code
  • Template literals for string interpolation
  • Spread operator for array/object manipulation
  • Default parameters for flexible functions
modern-js.js
// Arrow Functions & Destructuring const users = [ { name: 'Alice', age: 30 }, { name: 'Bob', age: 25 } ]; // Map with arrow function const names = users.map( ({ name }) => name ); // Template Literals const greet = (name) => `Hello, ${name}!`; // Spread & Rest const merge = (...arrays) => [...new Set(arrays.flat())];
